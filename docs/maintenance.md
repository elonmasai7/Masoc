# Ongoing Maintenance Strategy

## Daily
- Review SIEM critical/high alerts.
- Verify backup job success.
- Review failed authentication trends.

## Weekly
- Validate user and service accounts; disable unused accounts.
- Run vulnerability scans and triage findings.
- Review firewall/VPN rule drift.

## Monthly
- Patch operating systems, containers, and network firmware.
- Execute restore drill and document outcome.
- Review RBAC assignments with clinic leadership.
- Deliver a 15-minute phishing/security refresher.

## Quarterly
- Incident response tabletop exercise.
- Third-party/vendor access review.
- Validate immutable backup controls and retention.

## KPIs
- Mean time to detect (MTTD)
- Mean time to contain (MTTC)
- Patch compliance rate
- Backup success and restore success rates
- High-risk vulnerability closure time
